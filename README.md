# AI 623 – Assignment 0

This repository contains code and experiments for **Assignment 0**
**AI 623: Deep Vision Language Models**.

## Repository Structure

- `Task1.ipynb` – ResNet-152 transfer learning, ablation studies, and feature visualization  
- `Task2.ipynb` – Vision Transformer analysis, attention visualization, and robustness experiments  
- `Task3.ipynb` – Variational Autoencoder training and posterior collapse analysis  
- `Task4.ipynb` – CLIP zero-shot classification, modality gap analysis, and alignment experiments  

## Notes

- All models use pretrained weights unless otherwise specified.
- Experiments were conducted on STL-10 and FashionMNIST datasets where applicable.
- The repository is intended to support the accompanying written report.
